[Symfony 8](https://symfony.com) & [VueJS 3](https://vite.dev/) monolithique.

## Installation

Install dependencies Symfony 8 & VueJS 3

```bash
composer install
npm install
```

## Running the app

```bash
symfony serve
npm run vite
```
